Print · cut · fold — patient hand cards

  1. Main deck 01–14 then optional Backup 15–18 (marked). Letter · Actual size 100% · color on.
  2. Cut on dashed midlines. Bottom-right number = stack order.
  3. Fold the amber “Escalation” flap backward so only Presenting shows. Reveal when it escalates.
  4. 01–06 start Delayed (simpler). Most 07–14 start Urgent (Bee-kin Delayed). Quickling = 10.
  5. Also print kanban tokens. Treating quiets a problem for now — Fear can flare it; goal is keep people alive / Stable.
